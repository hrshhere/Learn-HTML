Form

The form element contains the entire form.

Action

The action attribute specifies where the form data will be sent upon submission (usually a server-side script that processes the data).

Method

The method attribute specifies the HTTP method to be used when submitting the form, often "post" for secure data transmission.

Label

Labels are used to describe each form control, enhancing accessibility.

Input

Form controls like text inputs Input, email inputs, and text areas textarea are provided for user input.

Required

The Required attribute on form controls ensures that the user must fill in the corresponding field before submission.

Submit

The submit button (input type="submit") triggers the form submission.
